# Scenario 1: Creating and Setting Up a Chama Group

**Persona**: Grace Wanjiku (Group Administrator)  
**Goal**: Set up a digital Chama for her women's market vendor group  
**Context**: Saturday afternoon, at home after market closes  
**Duration**: ~15 minutes  

---

## Background

Grace leads a women's savings group of 25 market vendors. They've been meeting monthly for 3 years, but manual record-keeping has become problematic. After hearing about Kijiji-Trust from another group admin, she decides to try the digital platform.

---

## Step-by-Step Scenario

###1. Discovery & Decision
- **Time**: Saturday, 3:00 PM
- **Location**: At home, children are playing outside
- Grace opens WhatsApp and finds the Kijiji-Trust link shared by her friend
- Clicks the link and lands on the website
- **Thought**: "Let me see if this is as simple as she said"

### 2. Registration
- **Time**: 3:02 PM
- Grace clicks "Get Started" button
- Fills in registration form:
  - Name: Grace Wanjiku
  - Phone: +254 712 345 678
  - Email: grace.wanjiku@gmail.com
  - Password: Creates a secure password
- Receives SMS verification code
- Enters code and account is created
- **Emotion**: Pleased it's straightforward

### 3. First Login & Welcome
- **Time**: 3:05 PM
- Automatically logged in after registration
- Sees welcome screen with simple tutorial
- Skips tutorial (will come back if needed)
- Lands on dashboard - empty state

### 4. Creating the Group
- **Time**: 3:06 PM
- Clicks large "Create New Group" button
- Fill in group details form:
  - **Group Name**: "Mama Mboga Welfare Group"
  - **Description**: "Market vendors mutual aid group"
  - **Contribution Amount**: 500 KES
  - **Contribution Frequency**: Monthly
  - **Purpose**: Selects "Bereavement & Medical Emergency"
- Reviews information
- Clicks "Create Group"
- **Success Message**: "Group created successfully!"

### 5. Getting the Invite Code
- **Time**: 3:09 PM
- System generates group code: **KT-MMW-2024**
- Grace sees options:
  - Copy code
  - Share via WhatsApp
  - Share via SMS
- She clicks "Share via WhatsApp"

### 6. Inviting Members
- **Time**: 3:10 PM
- WhatsApp opens with pre-filled message:
  > "Join our Mama Mboga Welfare Group on Kijiji-Trust! 
  > Use code: KT-MMW-2024
  > Monthly contribution: 500 KES
  > [Link to join]"
- Grace selects her vendors WhatsApp group (24 members)
- Sends invitation
- Returns to Kijiji-Trust app

### 7. Initial Configuration
- **Time**: 3:12 PM
- Sets herself as Admin (automatic)
- Configures emergency request rules:
  - Maximum request: 50,000 KES
  - Voting required: Yes (if over 20,000 KES)
  - Auto-approval: No
- Adds co-admin: Selects Mary (will add when she joins)

### 8. Exploring Features
- **Time**: 3:14 PM
- Views empty dashboard:
  - Current Balance: 0 KES
  - Total Members: 1 (herself)
  - Pending Contributions: 0
- **Thought**: "This looks clean and simple"
- Checks "How to Use" tutorial
- Bookmarks the page on her  phone

### 9. First Member Joins
- **Time**: 3:18 PM (a few minutes later)
- Push notification: "Jane Muthoni joined your group!"
- Grace opens app, sees:
  - Total Members: 2
  - Jane's profile appears in member list
- **Emotion**: Excited it's working!

---

## Key Moments

### ✅ Success Points:
1. Registration completed in under 3 minutes
2. Group created without confusion
3. Invite code easy to share via WhatsApp
4. No technical errors encountered
5. First member joined within 10 minutes

### ⚠️ Potential Pain Points:
1. Wanted to skip SMS verification (data costs)
2. Unsure about "voting threshold" setting
3. Wondered if she could edit group details later
4. Initially confused about member roles

---

## Outcomes

**Immediate**:
- ✅ Group "Mama Mboga Welfare" successfully created
- ✅ Invite code shared with 24 potential members
- ✅ First member joined within minutes
- ✅ Grace feels confident about the system

**24 Hours Later**:
- 18 out of 25 members have joined
- Grace has answered 3 WhatsApp questions about joining
- System feels manageable and useful

**1 Week Later**:
- All 25 members onboarded
- First contributions recorded
- Grace has spent 90% less time on admin work

---

## Design Validation

### HCI Principles Demonstrated:

1. **Simplicity**: Entire setup in ~15 minutes
2. **Visibility**: Clear progress indicators throughout
3. **Feedback**: Confirmation messages at each step
4. **Error Prevention**: Form validation prevented mistakes
5. **Familiar Patterns**: WhatsApp integration felt natural
6. **Mobile-First**: Completed entirely on smartphone
